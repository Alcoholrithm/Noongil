# Install Procedure

```
pip install flask opencv-python numpy

cd ./Oscar_Scripts/Oscar
Follow INSTALL.MD
```

```
cd ./Oscar_scripts/py-bottom-up-attention
python setup.py build develop
```

cd ./Oscar_sripts/scene_Graph_benchmark
